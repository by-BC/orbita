## 🌟 Sobre o Projeto

**Órbita** é uma aplicação web Fullstack construída com **Next.js 16**, utilizando o poder dos React Server Components, aliada ao **Supabase** para gerenciamento de banco de dados e autenticação de forma rápida e segura. A interface é desenhada utilizando **Tailwind CSS**, junto com **Lucide React** para ícones e **Recharts** para visualização de dados.

## 🚀 Como Executar Localmente

Siga os passos abaixo para rodar o projeto na sua máquina:

### 1. Clone o repositório
```bash
git clone https://github.com/by-BC/orbita.git
cd orbita
```

### 2. Instale as dependências
```bash
npm install
# ou
yarn install
# ou
pnpm install
```

### 3. Configuração de Variáveis de Ambiente
Crie um arquivo `.env.local` na raiz do projeto e preencha com as suas credenciais do Supabase:
```env
NEXT_PUBLIC_SUPABASE_URL=sua_url_aqui
NEXT_PUBLIC_SUPABASE_ANON_KEY=sua_chave_anon_aqui
```

### 4. Rode a aplicação
```bash
npm run dev
```
Abra [http://localhost:3000](http://localhost:3000) no seu navegador para ver o resultado! ✨
